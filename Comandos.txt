Comandos que usei no terminal e para o que serve:
  
  Comando : npm install -g expo-cli
  Por que : O Expo CLI facilita a criação e execução de projetos React Native.
  
  Comando : npx create-expo-app BibliotecaPessoal
  Por que : Cria a estrutura inicial do projeto "Biblioteca Pessoal".

  Comando : npx expo start
  Por que : Inicia o servidor do Expo para rodar o app no ​​emulador ou celular.
